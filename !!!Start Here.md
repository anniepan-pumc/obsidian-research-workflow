# Guidence
1. Create a Daily Note, read TODOs that wait for completion, or add new TODOs
2. To do a literature review, please first do the notation in Zotero
	1. Highlight the sentences and add comments in Zotero
		```
		**Zotero Highlight**
		### Yellow: Highlight
		### Red: Main topics; Aims
		### Green: Methods
		### Blue: Results
		### Purple: Dataset info
		### Magenta: Questions and Limitations
		### Orange: Supplementary Content (code; paper)
		### Grey: Unknown words/phrases
		```
	2. To move the literature note from Zotero to Obsidian, please do:
		`Right-click Zotero Item -> Obsidian Action -> Create Literature Notes`
	3. If there is any update, please do
		`Right-click Zotero Item -> Obsidian Action -> Update Literature Notes`
3. To do a research record, using 'Templater'
	1. Commend Line: `Templater: Create new note from template -> Research Note`
	2. Enter Title and Keywords
	3. A new note will be put in `Knowledge Summary/Research Records/**`
4. To summarize words and phrases, using 'Templater' as well
	1. Commend Line: `Templater: Create new note from template -> Word Explanation
	2. Enter Words and Fields
	3. A new note will be put in `Knowledge Summary/Words and Phrases/**`
